About Apache Flex UI Builder

Created by Igor Costa

Apache Flex UI Builder helps developers to create smooth UI without writing mxml in very basic approach

Since Adobe removed the Design Mode in Flash Builder, I used some html,js techninques to write a Visual mode

Keep in mind that this is just the begining, you can collaborate, fork, share and send pull requests
